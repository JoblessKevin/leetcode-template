# Bit Manipulation

## 1. 基礎運算子 (Basic Operators)

| 運算子 | 名稱 | 邏輯說明 | 範例 (二進位) |
| :--- | :--- | :--- | :--- |
| `&` | **AND (與)** | 兩者皆為 1，結果才為 1 | `1010 & 1100 = 1000` |
| `|` | **OR (或)** | 其中一個是 1，結果就是 1 | `1010 | 1100 = 1110` |
| `^` | **XOR (互斥或)** | **不同**則為 1，**相同**則為 0 | `1010 ^ 1100 = 0110` |
| `~` | **NOT (非)** | 0 變 1，1 變 0 (位元取反) | `~1010 = 0101` (簡化版) |
| `<<` | **左移** | 整體左移，右邊補 0 (相當於乘 $2^n$) | `0001 << 2 = 0100` (1 變 4) |
| `>>` | **右移** | 整體右移，保留正負號 (相當於除 $2^n$) | `0100 >> 1 = 0010` (4 變 2) |
| `>>>` | **無符號右移** | 整體右移，左邊一律補 0 | `-1 >>> 1` 會變成一個很大的正數 |

## 2. XOR 的魔術特性 (XOR Magic)

XOR 是位元運算中最常考的題目來源，請務必記住以下性質：
1. **自己跟自己 XOR 等於 0**：$x \wedge x = 0$
2. **任何數跟 0 XOR 等於自己**：$x \wedge 0 = x$
3. **交換律與結合律**：$a \wedge b \wedge a = (a \wedge a) \wedge b = 0 \wedge b = b$
   - *應用：找出陣列中唯一出現一次的數字（其他都出現兩次）。*

## 3. 常用位元技巧 (Common Bit Hacks)

### A. 檢查與操作特定位元 ($k$-th bit)
假設我們想操作數字 `n` 的第 `k` 位（從 0 開始算起）：

* **檢查第 k 位是否為 1**：
    `boolean isSet = (n & (1 << k)) != 0;`
* **將第 k 位設為 1 (Set bit)**：
    `n = n | (1 << k);`
* **將第 k 位設為 0 (Clear bit)**：
    `n = n & ~(1 << k);`
* **翻轉第 k 位 (Toggle bit)**：
    `n = n ^ (1 << k);`

### B. 判斷奇偶數 (Odd/Even)
不用 `% 2`，效率更高：
```java
if ((n & 1) == 0) // 偶數 (最後一位是 0)
if ((n & 1) == 1) // 奇數 (最後一位是 1)
```

### C. 移除最低位的 1 (Kernighan's Algorithm)
`n & (n - 1)` 可以快速將數字 `n` 最右邊的 `1` 變成 `0`。
* 範例：`n = 12 (1100)` -> `n - 1 = 11 (1011)` -> `1100 & 1011 = 1000 (8)`
* 應用：計算一個數字有多少個 `1` (Set Bits)，或判斷是否為 2 的冪次方。

### D. 檢查是否為 2 的冪次方 (Check if a number is a power of 2)
```java
boolean isPowerOfTwo = (n > 0) && ((n & (n - 1)) == 0);
```

### E. 取得最低位的 1 (Get the rightmost 1-bit)
`n & -n` 可以快速將數字 `n` 最右邊的 `1` 變成 `0`。
* 範例：`n = 12 (1100)` -> `n - 1 = 11 (1011)` -> `1100 & 1011 = 1000 (8)`
* 應用：計算一個數字有多少個 `1` (Set Bits)，或判斷是否為 2 的冪次方。

## 4. 常用算法

### Cyclic Sort
1. Range is [1, n] or [0, n]
2. Find the "missing", "duplicated", or "first positive" number.
3. Required O(n) time and O(1) space.
